## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/chagasleandro/teste.git
# Access
$ cd teste
# Install dependencies
$ yarn or npm 
# Run the project
$ yarn start or npm start 
# The server will initialize in the <http://localhost:3000>
